
usage: git flow <subcommand>

   init      Initialize a new git repo with support for the branching model.
   feature   Manage your feature branches.
   bugfix    Manage your bugfix branches.
   release   Manage your release branches.
   hotfix    Manage your hotfix branches.
   support   Manage your support branches.
   version   Shows version information.
   config    Manage your git-flow configuration.
   log       Show log deviating from base branch.

git branch  -   lista os branches
git status  -	exibe arquivos não comitados
git log	    -	exibe log de ações
git tag	    -   exibe as tags adicionadas	


CRIA NOVA FEATURE (local branch)
	 git flow feature start <name> [<base>]
	 EX:
	> git(develop):  git flow feature start welcome

	> git(feature/welcome):  git add .   (adiciona todos os arquivos)
				 git commit  -m 'added index' (adiciona arquivo index.html à feature, com comentário)	

FINALIZA A FEATURE
	> git(feature/welcome):  git flow feature finish welcome (finaliza a feature e faz merge para develop, volta para develop)
	> git(develop):
